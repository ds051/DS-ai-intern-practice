## 遇到的问题
输入Git命令时手误，生成了add、first-note.mod、first-note.modgit三个无用空文件；
提交前没有执行git status核对文件，直接全量提交，把垃圾文件上传到远程仓库。

## 问题总结&整改
提交代码前必须先执行 git status 查看待提交文件；
不要无脑使用 git add .，按需单独add需要的文件，避免无关文件污染仓库。
